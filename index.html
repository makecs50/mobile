Этот курс начинается там, где заканчивается Гарвардская программа CS50, переводя нас с веб-разработки в разработку мобильных приложений с использованием React Native. Это популярный open-source (проект с открытым исходным кодом) фреймворк, поддерживаемый Facebook'ом, позволяющий создавать кроссплатформенные, нативно отображаемые, приложения с помощью Javascript, без необходимости применения Java или Swift. Курс знакомит студентов с последней версией JavaScript (включая ES6 и ES7), а также с JSX - расширение синтаксиса JavaScript. Посредством реализации проектов, студенты приобретут опыт работы с React, его парадигмами, архитектурой приложения и пользовательскими интерфейсами. Кульминация данного курса будет в имлементации финального проекта, где студенты смогут с нуля написать приложение, на основе своего собственного дизайна.

Регистрация на https://www.extension.harvard.edu/academics/courses/mobile-application-development-react-native/25185.
